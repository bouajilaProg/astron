Terminal space scene with animated sprites, built with Go, Bubble Tea, and Lip Gloss.

![astron preview](https://placehold.co/800x400/080811/7EF9FF?text=astron+preview)

## Installation

```bash
go install github.com/bouajilaProg/astron@latest
```

Or download a prebuilt binary from the [Releases](https://github.com/bouajilaProg/astron/releases) page.

## Usage

```bash
astron
```

## Controls

- `q` or `ctrl+c`: quit
